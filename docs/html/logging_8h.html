Detailed Description

Logging interface.

The recommended fields: AMLIB_DOMAIN - the domain of the logging. "amlib" for amlib functions. MESSAGE - The human-readable message string. MESSAGE_ID - 128-bit identifier. Created using systemd-id128 new. PRIORITY - Syslog priority value. Set automatically by logging functions based on log level. CODE_FILE, CODE_LINE, CODE_FUNC - Source code info. ERRNO - value of errno.
